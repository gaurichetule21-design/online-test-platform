How it works

01

Build the test

Add multiple-choice, true/false, or short-answer questions and set a time limit.

02

Share the code

Send students a link or access code. They start as soon as they're ready.

03

Read the results

Features

Three question types

Multiple choice, true/false, and short answer — mix and match per test.

Timed attempts

A countdown keeps every student on the same clock, with auto-submit at zero.

Instant scoring

Objective questions are graded the second a test is submitted.

A dashboard for both sides

Instructors track attempts and averages; students keep a history of past scores.
